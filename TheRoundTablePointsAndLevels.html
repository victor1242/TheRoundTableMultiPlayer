Points and Levels

How XP is Earned (Per Round)

  • +2 participation
  • +1 if game has more than 2 players
  • +3 if you go out
  • +2 if you finish round rank #1

How Skill Points (SP) are Earned

Raw SP is the sum of several bonuses, then multiplied by the current room skill multiplier.

SP Gain: round(rawSkillPoints * skillMultiplier)

  • Rank bonus: 1st = +8, 2nd = +4, 3rd = +2
  • Pure set bonus: (pureSets * 2) + max(0, longestPureSet - 3)
  • Pure run bonus: (pureRuns * 3) + max(0, longestPureRun - 3)
  • Went out bonus: +4
  • No-assist bonus: +5 when all hints/warnings are off

Level Formula

progressScore = XP + (SP * 2)

Your level is based on progressScore and clamped from 0 to 9.
